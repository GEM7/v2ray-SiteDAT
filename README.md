# v2ray-SiteDAT

## 自定义site.dat 文件

### 下载

  - https://github.com/onplus/v2ray-SiteDAT/releases

### 使用：

  - 修改/添加sites文件夹下的域名文件，运行 ./v2sitedat 生成geosite.dat
  
  - 带路径 /v2sitedat -dat ./geofilepath/h2y.dat -dir ./geofilepath
    
  - 更多参考 ./v2sitedat --help
